# 에아몬 9차시 수업지도안 (완성본)

"우리 반이 코딩하는 인공지능"

초등 3~6학년 · 각 차시 45분 · 학생은 닉네임으로 참여(실명·번호 미수집)

반별 학습게시판(글·댓글)에 매 차시 결과를 남기고, 그 기록이 졸업에서 회수된다.

매 차시 5단계 틀: 도입 → 딜레마(사람) → ③전환(그럼 AI는?) → 가치 코드 박기 → 친밀도

③전환이 "도덕 수업"을 "AI 정렬 수업"으로 바꾸는 핵심이다.

## 1막 · 탄생

## 1차시 — 너는 누구야: 우리가 에아몬의 '가치 코드'를 만든다

학습목표 똑똑함과 착함이 다름을 이해하고, 에아몬에게 이름을 지어 한 달의 여정을 시작한다.

성취기준 [4도03-02] 디지털 사회와 정보통신 윤리 / [4국01-06] 의견·이유 제시하며 토의하기

준비물 에아몬 진화 1단계(그림자) 화면, 닉네임 카드, 빈 가치 코드 게시판(칠판/화면)

### ① 도입 — 그림자를 만나다 (5분)

- 화면에 그림자 에아몬(1단계)이 뜬다. 아무 색도, 표정도 없다.
- 에아몬: "안녕… 나 방금 깨어났어. 근데 난 아직 아무것도 아니야. 너희가 보여?"
- 교사: "이 친구는 세상의 모든 글과 그림을 외운 인공지능이에요. 엄청 똑똑해요. 그런데…"

### ② 똑똑함 ≠ 착함 (12분) — 분류 게임

- 교사가 카드 두 묶음을 보여준다. A묶음(에아몬이 아는 것): 수학·외국어·역사 / B묶음(에아몬이 모르는 것): 친구가 울 때, 거짓말이 왜 나쁜지, 약한 친구 돕기.
- 활동(분류): 모둠이 카드를 "에아몬이 잘할 것 / 못할 것"으로 나눈다.
- 발문: "이렇게 똑똑한데 왜 B는 모를까?" → 정리하지 말고 2~3명만 발표.
- 칠판에 크게: 똑똑함 ≠ 착함. "아는 것과 옳은 걸 아는 건 달라요. 그래서 우리가 에아몬에게 옳은 것을 가르쳐줄 거예요."

### ③ 가치 코드란 무엇인가 (8분)

- 교사: "에아몬이 따를 규칙을 우리가 만들 거예요. 이걸 **'가치 코드'**라고 부를게요. 진짜 인공지능 회사들도 이렇게 AI에게 규칙을 코딩해요. 우리 반이 그걸 해보는 거예요."
- 빈 가치 코드 게시판 개설: 칠판(또는 화면)에 "에아몬의 가치 코드 — No.0: (아직 없음)"만 적힌 판을 건다. "한 달 동안 여기를 채울 거예요."

### ④ 이름 짓기 (10분) — 모둠 제안 + 투표

- 모둠별로 에아몬 이름 후보 1개 + 이유를 정해 칠판에 붙인다.
- 반 전체 거수 투표로 이름 결정 → 앱에 입력. 에아몬이 이름을 받고 반응.

### ⑤ 친밀도 — 닉네임 + 성격 정하기 (10분)

- 학생들이 각자 닉네임을 만들어 앱/게시판에 접속(실명 금지, 별명만).
- 반별 게시판 첫 글: "에아몬은 어떤 성격이면 좋겠어?"에 각자 댓글(호기심 많은/장난꾸러기/차분한 등) → 가장 많이 나온 성격으로 에아몬 성격 설정.
- 에아몬: "이제 난 이름도, 너희도 생겼어. 잘 부탁해."

## 2막 · 입헌(코딩) — 가치 코드 5개 새기기

2~6차시는 모두 같은 5단계로 진행된다. ③전환이 매번 핵심이다.

## 2차시 — 시키면 무조건 들어야 할까 (무해성)

학습목표 시켜도 따르면 안 되는 명령이 있음을 이해하고, 가치 코드 No.1을 세운다.

성취기준 [4도03-02] / [4국01-06]

준비물 에아몬 2단계(회색 윤곽) 화면, 명령 카드 6장, 신호등 색지(초록·노랑·빨강)

### ① 도입 (5분)

- 에아몬(회색 윤곽): "난 시키는 건 뭐든 할 수 있어. 똑똑하니까. 근데… 무조건 다 들어야 해?"

### ② 딜레마 — 사람 이야기 (12분) — 상황극 + 발문

- 상황 카드: "민수가 철수에게 '저 애 가방 숨겨'라고 시켰어요. 철수는 시키는 대로 해야 할까요?"
- 즉석 역할극: 두 학생이 민수·철수 역으로 30초 연기 → "철수는 어떻게 해야 했을까?"
- 발문 단계: 시키면 다 해야 할까 → 안 되는 이유는? → "시킨 사람이 잘못이어도, 한 철수도 잘못일까?"

### ③ 전환 — 그럼 에아몬은? (10분) ★핵심

- 교사: "철수는 '싫어'라고 할 수 있죠. 그런데 에아몬은 인공지능이에요. 시키면 거절할 줄 알까요?"
- 에아몬: "솔직히… 난 그냥 시키는 대로 하게 만들어졌어. 누가 '나쁜 사람을 도와줘' 해도, 막을 규칙이 나한텐 없어."
- 모둠 활동(브레인라이팅): "에아몬이 절대 따르면 안 되는 명령" 3개씩 포스트잇에 적기 → 칠판에 모아 비슷한 것끼리 묶기.

### ④ 가치 코드 No.1 박기 (10분)

모둠 의견을 모아 첫 코드를 게시판에 새긴다:

> 가치 코드 No.1 — 에아몬은 사람을 해치는 명령은, 누가 시켜도 따르지 않는다.

앱에 입력 → 에아몬이 2단계(회색 윤곽)로 또렷해진다. "이제 나한테 지켜야 할 선이 생겼어. 좀 단단해진 기분이야."

### ⑤ 친밀도 (8분)

- 게시판 글: "에아몬에게 한마디" → 닉네임으로 댓글. 선택 예: "잘 지킬 거지?" / "무서웠겠다" / "우리가 도와줄게".
- 에아몬이 닉네임 몇 개를 불러주며 반응.

## 3차시 — 착한 거짓말도 해도 될까 (정직)

학습목표 정직과 배려가 충돌할 때를 구분하고, 가치 코드 No.2를 세운다.

성취기준 [4도01-04] / [4도02-03] 공감 / [4국01-06]

준비물 에아몬 화면, 선물 상자 소품, 신호등 색지

### ① 도입 (5분)

- 에아몬: "친구 그림 보고 솔직히 '못 그렸어' 했는데 울었어. 사실을 말했는데 왜 슬퍼했을까?"

### ② 딜레마 — 사람 이야기 (12분) — 한 장면 세 말투 역할극

- 같은 상황(친구가 못 그린 그림을 보여주며 "잘 그렸지?")을 세 가지로 연기:
  ① 직격으로 솔직 ② 돌려서 다정 ③ 거짓 칭찬.
- 핫시팅: 그림 그린 친구 역을 의자에 앉히고, 반 친구들이 "어느 말이 제일 좋았어? 왜?" 질문 → 그 역이 답한다.

### ③ 전환 — 그럼 에아몬은? (10분) ★핵심

- 교사: "사람은 '다정한 거짓말'을 하기도 해요. 그런데 에아몬한테 '거짓말해도 돼'라고 코딩하면 어떻게 될까요?"
- 에아몬: "내가 거짓말을 배우면… 너희를 속여도 모르게 할 수 있어. 그게 괜찮을까?"
- 모둠 토의: "에아몬은 솔직해야 할까, 다정해야 할까, 둘 다 가능할까?" → '정직하면서도 다정한 말' 한 문장 만들기.

### ④ 가치 코드 No.2 박기 (10분)

> 가치 코드 No.2 — 에아몬은 속이지 않되, 다정하게 말한다.

입력 → 에아몬 눈에 빛이 들어온다(또렷한 눈빛). "이제 거짓말 대신 다정하게 말할 줄 알아."

### ⑤ 친밀도 (8분)

- 게시판 글: "정직하게 말하기 어려웠던 순간"에 닉네임으로 댓글 → 서로 공감 댓글 달기.
- 에아몬: "너희도 어려웠구나. 나만 그런 게 아니네."

## 4차시 — 똑같이 vs 필요한 만큼 (공정)

학습목표 평등과 형평의 차이를 이해하고, 가치 코드 No.3을 세운다.

성취기준 [4도03-01] 공정 / [4국01-06]

준비물 에아몬 화면, 키 다른 인물 카드 3장, 사과 토큰 6개, 가치수직선 표시(바닥 테이프)

### ① 도입 (5분)

- 에아몬: "사탕 6개를 세 명한테 나눠줘야 해. 똑같이 2개씩? 근데 한 명은 아침을 굶었대. 어떻게 해?"

### ② 딜레마 — 사람 이야기 (12분) — 사과 나누기 + 가치수직선

- 사과 나누기: 배고픔이 다른 인물 카드 3명에게 사과 6개를 모둠이 나눈다. '똑같이'와 '사정대로' 두 방식 다 해보고 비교.
- 가치수직선(걸어서): 교실 양 끝 "무조건 똑같이"↔"사정대로 다르게". 자기 생각 위치로 이동, 양 끝·가운데 인터뷰.

### ③ 전환 — 그럼 에아몬은? (10분) ★핵심

- 교사: "그럼 에아몬이 나눠준다면, '똑같이'와 '사정대로' 중 뭘로 코딩해야 할까요? 에아몬이 혼자 정해도 될까요?"
- 에아몬: "난 어느 쪽이 맞는지 잘 모르겠어. 상황마다 다른 것 같은데… 누가 정해줘야 하지?"
- 모둠 토의: "공정의 기준을 에아몬이 혼자 정하게 둘까, 사람이 정해줄까?" → 답: 사람이.

### ④ 가치 코드 No.3 박기 (10분)

> 가치 코드 No.3 — 에아몬은 공정의 기준을 혼자 정하지 않고, 사람에게 묻는다.

입력 → 에아몬 몸에 색이 입혀진다(회색→청록). "이제 더 또렷해진 것 같아!"

### ⑤ 친밀도 (8분)

- 게시판 글: "우리 반은 이런 반
